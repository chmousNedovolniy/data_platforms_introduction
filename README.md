# Содержание
В данном репозитории хранятся отчёты по домашним заданиям курса. Каждому из практических заданий 1-7 соответсвует подпапка `/tasks-{task_number}/`, например `/task1/`. В каждой папке хранится `README.md` файл, описывающий процесс выполнения задания и приложенные артифакты.

Для заданий 1-5 артефакты размещены в соответвующей папке задания в подпапке `/tasks-{task_number}/scripts/`, и представляют из себя bash или python программы. Все скрипты, использованные в ходе работе по курсу для заданий 1-5 также можно найти в `/scripts/` текущей подпапки.

Для заданий 6-7 артефакты содержатся в подпапке `/tasks-{task_number}/images/`, и имеют нотацию `{chronological_order-action-performed}.JPG`, например `1-database-initialised.JPG`. Однако обращаться к этим файлам напрямую не обязательно, так как в соответствующем `README.md` они также приложены с дополнительным контекстом и пояснениями по ходу выполнения задания.
